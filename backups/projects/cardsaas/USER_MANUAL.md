# CardSaaS User Manual

This guide explains the full day-to-day flow for using CardSaaS as a workspace owner, admin, or card publisher.

## 1. What CardSaaS Does

CardSaaS lets you create and manage digital business cards, publish them as shareable web pages, collect leads, and control launch status from a private dashboard.

In the current setup:

- the public site lives at `https://v.2ai.az`
- the private workspace starts at `https://v.2ai.az/login`
- `Formag Corporate` is the main fixed corporate template
- manual activation is enabled, so new cards do not go live until they are approved

## 2. Create an Account

1. Open `https://v.2ai.az/register`
2. Enter:
   - `Name`
   - `Email`
   - `Password`
3. Submit the form
4. After registration, sign in at `https://v.2ai.az/login`

Notes:

- emails are normalized before authentication, so case differences do not break login
- if your email is listed in `ADMIN_EMAILS`, your account automatically gets admin access

## 3. Sign In

1. Open `https://v.2ai.az/login`
2. Enter your email and password
3. Click `Sign In`
4. You will be redirected to `/dashboard`

From the dashboard you can access:

- `Cards`
- `Leads`
- `Templates`
- `Team`
- `Admin` (for admin users)

## 4. Reset a Forgotten Password

1. On the login page, click `Forgot password?`
2. Open `https://v.2ai.az/forgot-password`
3. Enter your email address
4. Click the submit button
5. CardSaaS sends a password reset link through Resend
6. Open the link from your inbox
7. Set a new password
8. Return to login and sign in with the new password

Security behavior:

- the response is intentionally neutral, even if the email does not exist
- reset links are single-use
- passwords are stored as hashes, not plain text

## 5. Understand the Main Workspace

### Cards

This is the main control area for your digital cards.

You can:

- create a new card
- open a card
- edit a card
- delete a card
- see whether a card is `pending`, `active`, or `paused`

### Leads

This area collects inbound leads generated by public cards.

You can:

- see all leads
- filter and review lead information
- work through contact status

### Templates

This is where reusable visual templates live.

The current built-in template is:

- `Formag Corporate`

### Team

This area is for workspace collaboration and team management.

### Admin

Admin users can:

- approve cards
- pause cards
- review audit activity
- manage approval notes
- control owner-level activation state

## 6. Create a Card from the Formag Corporate Template

This is the main publishing flow.

1. Open `Dashboard -> Templates`
2. Find `Formag Corporate`
3. Click `Create card`
4. You will be taken to `/dashboard/cards/new`
5. Fill in the employee-specific fields
6. Save the card

The builder is already preconfigured for the Formag theme, so you do not need to manually rebuild the brand style every time.

## 7. Which Fields You Fill for a Formag Card

When creating a Formag card, focus on the employee data only.

Primary fields:

- `Profile Photo`
- `Full Name`
- `Job Title`
- `Primary Phone`
- `Secondary Phone`
- `Email`
- `Office Address`
- `URL Slug`
- optional `Custom Domain`

Brand-locked fields:

- company is fixed to `Formag Forwarding`
- website is fixed to `www.formag.com`
- theme is fixed to `formag_corporate`
- the visual direction and back side stay fixed

## 8. What Happens After You Save a Card

After creation, the card is stored in the workspace but does not necessarily go public immediately.

Because manual activation is enabled, the usual flow is:

1. card is created
2. card enters `pending`
3. admin reviews it
4. admin switches it to `active`
5. card becomes publicly accessible

This prevents unfinished cards from going live automatically.

## 9. Activate a Card

If you are an admin:

1. Open `Dashboard`
2. Find the new card
3. Use the activation control or go to the `Admin` section
4. Change the card state to `Active`

Possible card states:

- `pending` — waiting for approval
- `active` — public and live
- `paused` — intentionally blocked from public access

## 10. Open and Share a Public Card

Once a card is active, it can be opened through its public slug:

- `https://v.2ai.az/card/<slug>`

The public card supports:

- phone interaction
- email interaction
- office map opening
- website opening
- save to contact
- share card
- QR code
- front/back view switching for the Formag design

## 11. How the Public Card Behaves

The Formag card is mobile-first and opens as a polished digital business card.

On the front side the user sees:

- photo
- full name
- job title
- contact details
- save/share controls

On the back side the user sees:

- brand presentation
- QR code
- direct contact-save flow

## 12. What Each Interactive Action Does

### Phone

- opens the phone dial action
- if multiple numbers are configured, the UI can present a choice

### Email

- opens the default mail app using `mailto:`

### Office

- opens the office location in maps

### Website

- opens the company site in the browser

### Save Contact

- downloads or opens the generated `vCard` contact
- this is the same contact payload used by the QR destination

### Share Card

- uses the browser share flow where available
- otherwise falls back to copying the link

## 13. Edit an Existing Card

1. Open `Dashboard`
2. Find the card
3. Click `Edit`
4. Update the employee information
5. Save the changes

This is the right flow for:

- replacing a photo
- changing phone numbers
- updating email
- changing office address
- adjusting slug or public routing

## 14. Leads Flow

When someone interacts with the public card and submits lead information:

1. the lead is stored in the CardSaaS database
2. the lead appears in `Dashboard -> Leads`
3. optional integrations can forward the lead to external systems

Depending on the setup, lead notifications can also be sent by email.

## 15. Admin Flow

Admin users have the final approval layer.

Typical admin responsibilities:

- review newly created cards
- activate or pause cards
- add internal notes
- review owner-level state
- inspect audit activity

This allows the workspace to maintain quality control before public publishing.

## 16. Recommended Daily Workflow

For a normal operator:

1. Sign in
2. Open `Templates`
3. Choose `Formag Corporate`
4. Click `Create card`
5. Fill in the employee information
6. Save the card
7. Ask an admin to activate it, or activate it yourself if you are an admin
8. Open the public card and test:
   - phone
   - email
   - address
   - website
   - save contact
   - share card

## 17. Recommended Go-Live Checklist

Before sharing a card publicly, verify:

- the photo is correct
- the name is correct
- the title is correct
- the primary phone is correct
- the email is correct
- the office address is correct
- the slug is clean
- the card is `active`
- the public link opens correctly
- `Save contact` works
- the back side opens properly

## 18. Troubleshooting

### I can open the site, but I cannot enter the workspace

Check:

- correct email
- correct password
- reset password flow if needed

### My card exists, but it is not public

Most likely the card is still:

- `pending`
- or `paused`

An admin must switch it to `active`.

### I do not see Formag Corporate

Make sure you are signed in. System templates are available inside the authenticated workspace.

### Save Contact does not open exactly like a native app

This depends on the browser and the operating system. The system uses a standards-based vCard flow, which is the most compatible way to push contact data into mobile contact apps.
