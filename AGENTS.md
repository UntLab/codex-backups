# Локальные инструкции для Codex в этом проекте

## Язык
- Всегда отвечай пользователю на русском языке.

## Предпочитаемые скиллы
- Для задач по Python и анализу данных: `jupyter-notebook`.
- Для задач по веб-автоматизации и UI-потокам: `playwright`.
- Для задач по документам: `doc`, `google-docs`.
- Для таблиц и структурированных данных: `spreadsheet`.
- Для интеграций и workflow-автоматизации: `n8n`.
- Для задач по API OpenAI: `openai-docs`.

## Правило применения
- Если запрос явно совпадает с назначением скилла, сначала используй соответствующий скилл, затем выполняй задачу.

## Резервное копирование
- Если работа идёт прямо внутри этого репозитория, после завершения задачи напоминай пользователю безопасную команду публикации из корня репозитория:
  `bash scripts/sync/direct-push.sh main`
- Backup-копии отдельных проектов лежат в `backups/projects/` и не считаются основным местом разработки.
- Не предлагай `bash scripts/backup-push.sh` как основной путь для прямой работы в `codex`: этот сценарий зеркалит внешние папки через `rsync --delete`.
